# Welcome to Dermodel: an interactive skincare product/ingredient analysis platform with delightful user experience in mind

This project is a skincare analysis application, leveraging Supabase, React, and Spline to provide a 3 dimensional experience to visually supplement education

## Project Info

Dermodel is an independent project designed for comprehensive skincare analysis.

## Where can I try?

You can access the web app at www.dermodel.app

## What technologies are used for this project?

This project is built with:

* Vite
* TypeScript
* React
* shadcn/ui
* Tailwind CSS
